# devops-quiz-app
a static quiz app

## Question Format

The DevOps Question Bank is stored in `questions.json`.

Each question follows this structure:

- `id`: Unique identifier for the question (e.g., Q1, Q2)
- `topic`: DevOps topic category (e.g., Continuous Integration)
- `question`: The question text
- `options`: An array of multiple-choice options
- `answerIndex`: Index of the correct option (0-based)
- `explanation`: Explanation of why the answer is correct
